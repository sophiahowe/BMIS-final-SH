What is Tableau?

Tableau is a free software that enables users to better understand data sets. Tableau is very user-friendly and anyone is able to learn how to use the many features that Tableau offers. Tableau charts are "live" meaning that they are continuously being updated as more information become available.

Interested in learning more? Check out Tableau's website.
